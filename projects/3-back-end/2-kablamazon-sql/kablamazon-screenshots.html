Kablamazon

The Kablamazon Application

Kablamazon is a command line application that functions as a storefront using a MySQL database.


How to Use

To use the app, you must have Node installed, and must access it via the command line.


After installing Node, go to the root directory of the application, where the package.json file is located,
and npm install to install the required dependencies.


To start the app run:

node kablamazonCustomer.js

Then follow the directions on screen!
